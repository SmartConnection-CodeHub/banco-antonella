> [!gantt] roadmap-gantt
> Hitos por semana · owner explícito · dependencias visibles.

**Lunes**
- Sesión 1 con Antonella: validar Ruta del [doc 02](02-tension-ds19.html) · firmar OK
- Validar PIN del banco · enviar URL GitHub Pages a Anto + Rommina

**Martes**
- Anto agenda asesoría DS19 con MINVU (gratis · oficina Viña)
- Guillermo verifica disponibilidad cotizaciones referenciales para cuadros

**Miércoles**
- Asesoría MINVU: confirmar elegibilidad DS19 · timing · documentos faltantes

- Llena con Anto los datos faltantes: rol mamá Rommina · contactos proveedores · dirección local potencial

## Semana 2 (días 8–14) · Dossier maestro

**Lunes-Martes**
